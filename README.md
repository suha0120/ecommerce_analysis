# ecommerce_analysis

# 🛒 Ecommerce Sales Analysis Project

## 📌 Project Overview

This project focuses on analyzing an ecommerce database to extract
meaningful business insights related to **sales performance, customer
behavior, product trends, and regional analysis**.

------------------------------------------------------------------------

## 🎯 Objective

To analyze and answer key business questions such as: - How much revenue
is generated? - Which products perform best? - What is customer behavior
and segmentation? - How do returns impact business performance?

------------------------------------------------------------------------

## 🗂️ Database Schema

The project is based on the following tables: - Customers\
- Products\
- Orders\
- OrderDetails\
- Regions

------------------------------------------------------------------------

## 📊 Analysis Performed

### 🔹 1. General Sales Insights

-   Total Revenue Generated\
-   Revenue (excluding returned orders)\
-   Monthly & Yearly Revenue Trends\
-   Revenue by Product & Category\
-   Average Order Value (AOV)

------------------------------------------------------------------------

### 🔹 2. Customer Insights

-   Top 10 Customers by Revenue\
-   Repeat Customer Rate\
-   Customer Segmentation:
    -   Platinum (\>1500)
    -   Gold (1000--1500)
    -   Silver (500--999)
    -   Bronze (\<500)
-   Customer Lifetime Value (CLV)\
-   Average time between orders

------------------------------------------------------------------------

### 🔹 3. Product & Order Insights

-   Top 10 Products (by Quantity & Revenue)\
-   Return Rate per Product\
-   Return Rate by Category\
-   Average Product Price per Region\
-   Sales Trends by Category

------------------------------------------------------------------------

### 🔹 4. Regional Insights

-   Order Volume by Region\
-   Revenue Distribution by Region

------------------------------------------------------------------------

### 🔹 5. Return & Refund Insights

-   Overall Return Rate\
-   Return Rate by Region\
-   Customers with Frequent Returns

------------------------------------------------------------------------

## 🧠 SQL Concepts Used

-   Joins (INNER JOIN)
-   Aggregations (SUM, AVG)
-   GROUP BY & ORDER BY
-   CTEs (Common Table Expressions)
-   Window Functions (LEAD, LAG)
-   Conditional Logic (CASE)

------------------------------------------------------------------------

## 📈 Key Metrics

-   Revenue\
-   Average Order Value (AOV)\
-   Customer Lifetime Value (CLV)\
-   Return Rate

------------------------------------------------------------------------

## 🚀 Conclusion

This project demonstrates how raw ecommerce data can be transformed into
actionable insights using SQL. It helps businesses: - Improve product
performance\
- Understand customer behavior\
- Optimize sales strategies\
- Reduce return rates
